# Conversor de PDF
### Opção com Interface Disponível para Windows e Linux

### Funções:
De PDF para DOCX
De PDF para DOC
Mais opções em breve...

### Bibliotecas usadas:
```
pip install pdf2docx python-docx
```

```
pip install PyQt5
```

### Como usar:
Windows:
```
python conversor.py
```
Linux:
```
python3 conversor.py
```
